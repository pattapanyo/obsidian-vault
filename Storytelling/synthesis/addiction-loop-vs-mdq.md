# Addiction Loop vs Major Dramatic Question / Three-Act

Short comparison mapping the four-step addiction loop (Stakes → Big Question → Head Fake → Rehook) onto the Major Dramatic Question (MDQ) and three-act beats.

- Stakes → Setup / Act 1: Establish what matters (context that creates the MDQ). See [[concepts/stakes]] and [[concepts/major-dramatic-question]].
- Big Question → Inciting Incident → MDQ: The explicit Yes/No query that drives Acts 2–3.
- Head Fake → Midpoint / False Payoff: A plausible but misleading payoff that delays the MDQ answer and increases tension.
- Rehook → New Complication / Act 2 Rising Action: Re-engages curiosity by escalating stakes or revealing new information; primes the next loop.

Practical notes
- Use the loop at scene/sequence level to sustain attention while keeping the MDQ intact at the story-level.
- Head fakes must be earned; overuse blunts trust. Prefer structural justification (character choice, new evidence).
- The loop is compatible with three-act escalation: each act can host multiple loops that raise stakes toward the climax.
